# Usage Guide

This document describes how to use FreeCVcreator in detail.

## Table of Contents

1. [Installation](#installation)
2. [Configuration](#configuration)
3. [Working with Templates](#working-with-templates)
4. [Exporting](#exporting)

## Installation

Install dependencies with:

```
npm install
```

## Configuration

Configuration files are located under `src/environments`. Modify them to connect to your own Firebase instance if needed.

## Working with Templates

Choose a template from the templates list. The editor supports rich text for profiles and job descriptions. You can add multiple jobs and skills.

## Exporting

When you are satisfied, click *Export PDF* to generate a download.
